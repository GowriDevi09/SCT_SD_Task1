# 🌡 Temperature Converter

A modern, responsive Temperature Converter web application developed using HTML, CSS, and JavaScript.

This project was created for the **SkillCraft Technology Software Engineering Internship**.

---

## 🚀 Features

- Convert between Celsius, Fahrenheit, and Kelvin
- Instant conversion
- Auto-convert while typing
- Swap units
- Copy result
- Dark / Light mode
- Local Storage support
- Conversion history
- Download history as CSV
- Clear history
- Absolute zero validation
- Responsive UI
- Keyboard shortcut (Enter)
- Loading animation
- Toast notifications

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ▶️ Run the Project

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter a temperature.
4. Select source and destination units.
5. Click **Convert**.
